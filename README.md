Baidu Translator in Command Line
===

A simple English-Chinese translator can be used in terminal. 

Installation
---
Download this package and  
```
python setup.py install
```   
or run   
```
(sudo) pip install baidu
```   
in terminal. 

Usage
---
```
baidu [word_to_be_translated]
```
